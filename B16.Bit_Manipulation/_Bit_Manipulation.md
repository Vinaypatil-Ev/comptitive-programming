# Bit Manipulation

#### Bitwise operators

|   Op Name   | Opearator |
| :---------: | :-------: |
|     And     |     &     |
|     Or     |           |
|     Not     |     ~     |
|     XOR     |     ^     |
| Left shift |    >>    |
| Right shift |    <<    |

#### **All about bit maupulation**

##### 1. Get i(th) bit

Num & (1 << i)

##### 2. Set i(th) bit

Num | (1 << i)

##### 3. Clear i(th) bit

Num & (~( 1 << i ))
